# 🎉 Birthday Invitation Website

A simple and responsive Birthday Invitation webpage created using **HTML5**. This project displays all the essential details of a birthday celebration, including the event date, time, venue, Google Maps location, and a list of things to bring.

---

## 🚀 Features

- 🎂 Beautiful birthday invitation page
- 📅 Displays event date and time
- 📍 Shows venue details
- 🗺️ Includes Google Maps location
- 🎁 List of things to bring
- 💻 Beginner-friendly HTML project

---

## 🛠️ Technologies Used

- HTML5
